# SteamVR-Photon
<hr>
- steamVR에 Photon을 이용하여 멀티 플레이를 구현
- VR Avatar 만들기 프로젝트가 추가됨
<br>

## 주의사항
<hr>
1. SteamVR Plugin은 받고 따로 설치해야함

## 나의 메모

2. RobotKyle 에셋의 Texture는 모두 ignore됨
